PhoneGap Release Notes¶

Version 3.3.3 (May 11, 2018)¶

New

  • Updated the native iOS SDK to 5.7.3. Please see Apple Release Notes for more details.

Version 3.3.0 (Feb 16, 2017)¶

New

  • Updated the native Android SDK to 5.8.4. Please see Android Release Notes for more details.

Fixes

  • Some plugin methods were not properly synchronized in Android.

Version 3.2.1 (Jan 25, 2017)¶

New

  • Updated the native Android SDK to 5.8.3. Please see Android Release Notes for more details.

Version 3.1 (Mar 1, 2016)¶

New

  • This is the first packaged release of the PhoneGap plugin that is available on npm.

  • Apteligent now requires explicit initialization using the Crittercism.init() method. Please refer to Installing the Workspace ONE Intelligence Cordova Plugin for installation instructions.

Removed

  • Previously iOS and Android app IDs were specified when running cordova plugin add. App IDs that were specified in this manner will be ignored in this release. Use the Crittercism.init() method mentioned above.

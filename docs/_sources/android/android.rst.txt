Android SDK
===========

Introduction
------------

This topic describes how to use Workspace ONE Intelligence with Android apps.
This guide assumes you have an Workspace ONE Intelligence account set up with a
valid app ID.

The Android SDKs may be downloaded :ref:`here <downloads>`.

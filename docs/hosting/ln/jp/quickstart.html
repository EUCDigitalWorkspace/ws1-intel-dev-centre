Crittercism クイックスタート¶

本クイックスタートでは、Crittercism ソリューションの実装を支援し、全機能の概要を説明します。これは、総合的なオンライン ドキュメンテーションに代わるものではありません。実装にかかる推定時間も記載されています。

ステップ 1:サインアップ (1 分)¶

無料の Crittercism アカウントにサインアップ します。これでシステム内にユーザーの認証情報が作成されます。

アプリケーションへの招待を受けた場合には、サインイン後にアプリケーションのパフォーマンス情報にアクセスできます。

ステップ 2:アプリの登録 (1 分)¶

アプリの登録 を行います (必要な場合)。 登録すると、Crittercism システム内でそのアプリケーションのオーナーとなります。

ステップ 3:Crittercism SDK のダウンロード (5 分)¶

Crittercism の [Downloads (ダウンロード)]ページ に移動し、 アプリ プラットフォームの Crittercism SDK をダウンロードします。

ステップ 6:シンボリケーションの追加 (5 分)¶

Crittercism は、自動的にサーバー側のシンボリケーションを使用して人間が読めるエラー レポートを作成します。 アプリのプラットフォームの手順に従って、アプリのシンボル ファイルを提供する必要があります。

Note

この時点で、基本的なアプリのロードのロギングとクラッシュ レポートが実装されています。

コラボレーターの招待 (1 分)¶

アプリケーションにコラボレーターを招待します。アプリを選択し、[App Settings (アプリ設定)] (左ペイン) に続いて [Team (チーム)] (右ペイン) を選択します。招待を受けた人は、電子メールによるアプリケーションへの招待の通知を受信します。

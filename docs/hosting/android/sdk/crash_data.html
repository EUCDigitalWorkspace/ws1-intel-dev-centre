CrashData¶

Import com.crittercism.app package to use this class.

CrashData provides information about crash that happens in the last session. See Detecting a Crash Occurred and CrittercismCallback for more details

Initialization¶

CrashData (name, reason, timeOccured)

Creates a CrashData object with specified name, reason, and time of the crash

public CrashData (String name, String reason, Date timeOccurred)

name

A String that describes the crash name

reason

A String that describes the crash reason

timeOccured

A Date that informs the time of crash
